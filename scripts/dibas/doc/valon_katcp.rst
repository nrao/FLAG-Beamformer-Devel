ValonKATCP
==========

.. toctree::
   :maxdepth: 1

The *ValonKATCP* class allows the Player to control the Valon synthesizer via a KATCP client. This is necessary as the Valon synthesizer is controlled by a serial port on the ROACH PPC host.
